Model
The model performs action recognition in 2 stages, action localization and classification. Stage 1: Action Localization Stage 2: Action Classification
Experiments
We are using the CalMS21 to test this architecture. Specifically their Task 1 and Task 3. Task 1 looks at 4 behaviors across 70 videos, and Task 3 looks at 7 more action over a smaller set of videos. All steps of the pipeline will be evalutated.
